# Zlib-ng

## About
[Zlib-ng](https://github.com/Dead2/zlib-ng) is an improvement of zlZlibib for new generations.

## Options
### To compress
* **Compression level** - (integer, 0-9, default 0)
  * **0** - obtains the fastest compression.
  * **9** - obtains the highest compression ratio.
