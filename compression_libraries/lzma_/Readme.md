# Lzma

## About
[Lzma](https://tukaani.org/xz/) is a compression library with an API similar to that of zlib.

## Options
### To compress
* **Mode** - (integer, 0-2, default 0)
  * **0** - Default.
  * **1** - Extreme.
  * **2** - Lzma2.
* **Threads** - (integer, 1-8, default 1)
  * Number of threads used by the compression library
